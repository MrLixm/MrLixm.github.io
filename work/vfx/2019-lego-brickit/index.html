Lego BrickIt models

05 January 2019

I bought the BrickIt plugin for Maya from Wizix and made those models. The plugin provides a library of parts and you have to assemble the model yourself by following the official lego instructions. Very fun to use, had a blast building those.

Maya and Arnold for the renders. Mix of Nuke and After effect for post-processing.
